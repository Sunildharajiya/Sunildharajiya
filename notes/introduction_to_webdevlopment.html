Web Development Basics

1. What is Web Development?

Web development is the process of creating, designing, and structuring websites.

In simple words, building a website is called web development.

2. What is a Website?

A website is a collection of multiple web pages connected under a common domain name.

Web page: A digital page on the internet that may contain text, images, videos, buttons, links, or files.

3. Basics of Networking

  • Network: A network is the connection of two or more computers to share data.
  • Internet: A global system of interconnected networks connecting millions of devices worldwide.
  • WWW (World Wide Web): Allows users to access and browse websites. Think of the WWW as a huge library, where each website is like a book.

4. Types of Websites

  1. Static Websites
    • Show the same content to every visitor.
    • Content does not change based on user actions.
    • Example: Portfolio websites, company info pages.
  2. Dynamic Websites
    • Content changes according to user actions or stored data.
    • Example: YouTube, Instagram, Amazon.

5. Stages of Web Development

There are two major stages:

(a) Front-End Development
  • This is the visible part of a website.
  • Includes design, layout, images, videos, buttons, and text.
  • Technologies: HTML, CSS, JavaScript.
(b) Back-End Development
  • The hidden part of a website users cannot see.
  • Handles logic, security, payment processing, and databases.
  • Languages: Python, PHP, Node.js.
  • Databases: MySQL, SQL, MongoDB.

Example: On Amazon, when we click Buy, backend processes payment and order confirmation.

6. Summary

  • Web Development = Building websites.
  • Website = Collection of web pages.
  • Web Pages contain text, images, videos, etc.
  • WWW = Global library of websites.
  • Two types of websites = Static & Dynamic.
  • Two stages of development = Front-End & Back-End.
